# Safe Third-Party Authority Checklist

Use the phrase "safe third-party authority" in client-facing language. Keep "parasite SEO" as internal shorthand only if needed.

## Before Creating Any Asset

- [ ] Platform is approved.
- [ ] Asset has a real buyer/search purpose.
- [ ] Source facts are approved.
- [ ] Link target is approved.
- [ ] Risk level is documented.
- [ ] Indexation/tracking row exists.

## Asset Quality Rules

- [ ] Content is useful and specific to TraxSolutions/nFocus.
- [ ] Claims are source-backed.
- [ ] Copy is unique enough for the platform.
- [ ] Links are natural and transparent.
- [ ] Images use approved brand/product facts.
- [ ] Community/platform rules are respected.

## Indexation And Interlinking

- [ ] Published URL captured.
- [ ] Linked target captured.
- [ ] Indexation status checked.
- [ ] Evidence URL or screenshot captured.
- [ ] Non-indexed URLs have next actions.
- [ ] Internal/third-party links do not create a spam footprint.

## Hard No

- [ ] No fake identities.
- [ ] No fake reviews.
- [ ] No fake testimonials.
- [ ] No fake press releases.
- [ ] No mass-spun duplicate content.
- [ ] No Reddit/community spam.
- [ ] No hidden or misleading claims.
- [ ] No guaranteed page-one or AI-citation promises.
