# 8-Week SEO + Entity Authority Implementation Plan

## Week 1: Audit And Map

Owner: Tony/Emma/SEO operator

- Run entity SEO audit.
- Build canonical fact map.
- Build buyer prompt map.
- Build query fanout map.
- Identify profile, review, directory, and third-party authority gaps.
- Create URL/indexation tracker.

Acceptance criteria:

- Canonical entity facts are ready for nFocus approval.
- Prompt/fanout map covers awareness, consideration, decision, and buyer-risk questions.
- No sensitive public facts are treated as approved unless sourced and cleared.

## Week 2: Owned-Site Foundation

Owner: SEO/content/web, after approval

- Update approved product/entity language.
- Plan buyer-risk FAQ or answer-first decision content.
- Plan schema tied to visible page copy.
- Plan internal links from product, proof, category, and comparison pages.
- Plan image SEO fixes.

Acceptance criteria:

- All public wording is source-backed or client-approved.
- Schema mirrors visible copy.
- No pricing/security/support/API claims are invented.

## Week 3: Profile Stack

Owner: client/marketing/SEO

- Audit LinkedIn, YouTube, review platforms, directories, and association/listing surfaces.
- Draft approved bio/description language.
- Validate sameAs candidates.
- Flag profiles with wrong, thin, or unapproved facts.

Acceptance criteria:

- Profile stack plan is approved before changes.
- Each profile has owner, status, risk level, and next action.

## Weeks 4-5: Safe Off-Site Authority Assets

Owner: content/marketing/SEO

- Produce approved asset briefs for LinkedIn, YouTube, Medium/Substack, Webflow/Ghost, directories, partner pages, or real-news surfaces.
- Create branded image pack briefs.
- Map links and target pages.
- Track planned URLs and evidence requirements.

Acceptance criteria:

- Every asset has a real buyer purpose.
- No fake reviews, fake press, fake identities, mass-spun content, or platform abuse.
- Nothing is published without explicit approval.

## Week 6: Indexing And Reinforcement

Owner: SEO/operator

- Confirm published approved URLs.
- Track indexation status.
- Add approved links and supporting mentions.
- Refresh or flag non-indexed assets.

Acceptance criteria:

- Each URL has indexation status, evidence, and next action.
- Unindexed assets are documented instead of hand-waved.

## Week 7: AI/GEO Testing

Owner: SEO/GEO operator

- Run approved prompt checks.
- Inspect citations and source use.
- Compare new source behavior against Day 1 baseline.
- Identify missing assets or weak citation paths.

Acceptance criteria:

- Prompt/citation results are evidence-backed.
- Findings are framed as selected-surface behavior, not universal truth.

## Week 8: Report And Next Push

Owner: Emma/Tony/SEO operator

- Produce final deliverables inventory.
- Report indexation, SERP movement, AI citation evidence, blockers, and next actions.
- Recommend the next 30-day authority push.

Acceptance criteria:

- Final report separates completed work, evidence, blockers, and recommendations.
- No ranking or AI-citation guarantees are made.
