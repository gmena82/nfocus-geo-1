# Platform Asset Matrix

| Platform | Asset type | Purpose | Link policy | Risk | Example use |
|---|---|---|---|---|---|
| nFocus owned site | Product, FAQ, proof, comparison, category pages | Canonical facts and citation targets | Internal links to proof and modules | Low if approved | TraxSolutions buyer-risk FAQ and nonprofit outcome-measurement page |
| LinkedIn company page | Profile update, posts, article | Entity consistency and buyer proof | Link to product/proof pages | Medium; needs client approval | Company description aligned with approved TraxSolutions positioning |
| LinkedIn personal profiles | Bio/profile language | Founder/leadership entity support | Link to company/product only if appropriate | Medium; personal approval needed | Leadership bio mentions nFocus/Trax category accurately |
| YouTube | Video, channel profile, description | Searchable explainer and proof surface | Link to owned product/proof pages | Medium; needs content approval | "How nonprofits track attendance and outcomes" explainer |
| Review platforms | G2, Capterra, Software Advice, GetApp, TrustRadius | Third-party trust/review corroboration | Profile links only where supported | High; facts/reviews must be real | Correct product category and approved description |
| Industry directories | Nonprofit/public-sector software listings | Entity/category corroboration | Link to canonical page | Medium | Reputable nonprofit software directory profile |
| Associations/partners | Partner page, member profile, resource mention | High-trust corroboration | Link to relevant owned source | Medium | Approved partner or association listing |
| Medium/Substack | Useful article | Searchable third-party content | Link naturally to owned sources | Medium | Article on outcome measurement questions for nonprofits |
| Webflow/Ghost | Supporting page/site | High-authority supporting asset if useful | Link to canonical sources | Medium-high; avoid thin pages | Approved topic page with unique content and proof links |
| Reddit/community | Comment or post plan | Authentic participation only | Link only if allowed and helpful | High; spam risk | Answer a real nonprofit software question without promotion |
| Facebook/X | Post/thread | Brand/entity and topical signal | Link naturally to owned pages | Medium | Approved thought-leadership post with visual asset |
| Press/news | Real announcement | Legitimate news corroboration | Link to official announcement | High; no fake press | Real client-approved milestone or report release |
| Cloud/listicle pages | Supporting page | Use only if quality-controlled and defensible | Link transparently | High; can become spam | Avoid unless Tony/client explicitly approves |

## Exclusions

- Fake reviews.
- Fake testimonials.
- Fake press.
- Fake identities.
- Spun duplicate content.
- Platform-rule evasion.
- Disposable link pages with no buyer value.
